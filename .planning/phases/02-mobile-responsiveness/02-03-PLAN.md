    Apply the following three conditional fixes based on the research findings. All three are known risk areas that the research identified as likely to cause visual issues at 375px. Per D-02, any mobile issue found is fixed in this phase.

    FIX 1 — Home heading mobile padding (src/style.css):
    The existing `.heading` and `.heading-name` rules at ~lines 323-330 each have `padding-left: 45px !important` with no mobile override. At 375px this takes up ~12% of viewport width. Add the following mobile override to style.css:

      @media (max-width: 767px) {
        .heading,
        .heading-name {
          padding-left: 15px !important;
        }
      }

    FIX 2 — Torus video height (src/components/Projects/Robotics/Torus.js + src/style.css):
    The video element at ~lines 51-58 has `height: "500px"` inline. At 375px this is taller than the viewport above the fold. Add `className="torus-video"` to the video element (keeping all existing inline style props), then add the CSS override to style.css:

      @media (max-width: 767px) {
        .torus-video {
          height: 250px;
        }
      }

    The video element currently has: `style={{ width: "100%", height: "500px", objectFit: "contain", borderRadius: "12px", display: "block" }}`. After: add `className="torus-video"` alongside the existing style prop. The CSS rule overrides height at mobile; all other inline style props stay.

    FIX 3 — ArtEngineering 3-col collage (src/components/Projects/Robotics/ArtEngineering.js + src/style.css):
    The div at ~lines 85-90 has `style={{ display: "grid", gridTemplateColumns: "1fr 1fr 1fr", gap: "8px", alignItems: "center" }}`. At 375px, three ~110px columns may be too small to be useful. Replace the inline grid props with `className="art-collage"` and keep only the non-grid props inline:

      className="art-collage" style={{ alignItems: "center" }}

    Add the CSS rule to style.css:

      .art-collage {
        display: grid;
        grid-template-columns: 1fr 1fr 1fr;
        gap: 8px;
      }

      @media (max-width: 767px) {
        .art-collage {
          grid-template-columns: 1fr;
        }
      }

    All new media queries must use `max-width: 767px` per D-06. Append all new CSS rules to style.css after the existing rules, grouped with a comment such as `/* === MOBILE FIXES (Phase 02) === */`.
  </action>
